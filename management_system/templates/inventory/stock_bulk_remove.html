Bulk Stock Removal

Warning:

This action will permanently deduct stock quantities from your inventory. Each removal will be recorded as a stock-out transaction.

Process:
  1. Download the template and fill in the items to remove
  2. Upload the completed Excel file
  3. System will validate each item and quantity
  4. Items with insufficient stock will be skipped
  5. Successful removals will create stock-out transactions
Column Required Description Validation Example
item_code Yes Item code (must exist in system) Must match existing item code ITEM-001
quantity Yes Quantity to remove Must be positive number ≤ available stock 10
remarks No Reason for removal Optional text Used for Project ABC
Tips:
  • Check stock levels before preparing the file
  • Use the Inventory list to verify item codes
  • Test with a small batch first
  • Keep a backup of your Excel file
